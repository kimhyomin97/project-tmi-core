# 99. 공통 기능 (common 패키지)

> `com.tmi.core.common` 아래에 두는 공통 클래스 카탈로그.

---

## 1. 패키지 구조

```
com.tmi.core.common
├── config/
│   ├── JpaAuditingConfig
│   └── FeignConfig
├── domain/
│   └── BaseEntity
└── exception/
    ├── IErrorType
    ├── ECommonErrorType
    ├── BaseException
    ├── GlobalExceptionHandler

```

## 2. 클래스

### 2.1 BaseEntity

**위치**: `common/domain/BaseEntity.java`

모든 엔티티의 공통 컬럼.

| 항목 | 내용 |
|---|---|
| 종류 | `abstract class` |
| 어노테이션 | `@MappedSuperclass`, `@Getter`, `@EntityListeners(AuditingEntityListener.class)` |
| 필드 | `createdAt`, `updatedAt`, `deletedAt` (`Instant`) |
| 자동 갱신 | `createdAt` = `@CreatedDate`, `updatedAt` = `@LastModifiedDate` |
| `deletedAt` | 자동 갱신 X. 서비스에서 명시적으로 `now()` 세팅 |

**컬럼 매핑**

| 필드 | 컬럼 | 옵션 |
|---|---|---|
| `createdAt` | `created_at` | `nullable=false`, `updatable=false` |
| `updatedAt` | `updated_at` | `nullable=false` |
| `deletedAt` | `deleted_at` | `nullable=true` |

id 는 두지 않음. 각 엔티티가 본인 PK 직접 선언.

### 2.2 JpaAuditingConfig

**위치**: `common/config/JpaAuditingConfig.java`

Spring Data Auditing 활성화 — `@CreatedDate` · `@LastModifiedDate` 어노테이션이 동작하도록 스위치만 켜는 설정.

| 항목 | 내용 |
|---|---|
| 어노테이션 | `@Configuration`, `@EnableJpaAuditing` |
| 본문 | 빈 클래스 |

### 2.3 FeignConfig

**위치**: `common/config/FeignConfig.java`

OpenFeign 활성화 + 모든 Feign client 에 공통 적용할 타임아웃·인터셉터·에러 디코더.

| 항목 | 내용 |
|---|---|
| 어노테이션 | `@Configuration`, `@EnableFeignClients(basePackages = "com.tmi.core.infra")` |
| 공통 설정 | connect timeout 1s, read timeout 2s |
| 에러 디코더 | 4xx / 5xx 응답을 도메인 예외로 변환 |

개별 client (예: `SimilarityClient`) 는 여기 두지 않음. `infra/{시스템명}/` 에 인터페이스로 선언.

### 2.4 IErrorType

**위치**: `common/exception/IErrorType.java`

모든 ErrorType enum 의 공통 계약.

| 항목 | 내용 |
|---|---|
| 종류 | `interface` |
| 메서드 | `String getErrorCode()`, `HttpStatus getStatus()` |

### 2.5 ECommonErrorType

**위치**: `common/exception/ECommonErrorType.java`

도메인에 속하지 않는 범용 에러 코드.

| 항목 | 내용 |
|---|---|
| 종류 | `enum implements IErrorType` |
| 어노테이션 | `@RequiredArgsConstructor`, `@Getter` |
| 필드 | `HttpStatus status`, `String errorCode` |

| 상수 | HTTP | 코드 |
|---|---|---|
| `INVALID_REQUEST` | 400 | `TMI.COMMON.INVALID_REQUEST` |
| `INTERNAL_ERROR` | 500 | `TMI.COMMON.INTERNAL_ERROR` |

도메인 에러 (`SENTENCE_*` 등) 는 각 도메인 패키지의 자체 enum 에 정의.

### 2.6 BaseException

**위치**: `common/exception/BaseException.java`

모든 도메인 예외의 추상 부모.

| 항목 | 내용 |
|---|---|
| 종류 | `abstract class extends RuntimeException` |
| 필드 | `IErrorType errorType`, `Object[] args` |
| 생성자 | `protected BaseException(IErrorType errorType, Object... args)` |
| HTTP 상태 | `getStatus()` → `errorType.getStatus()` 위임 |

각 도메인은 본인 패키지에 1 개의 예외 클래스만 둠 (예: `SentenceException`). 케이스는 도메인 ErrorType enum 으로 구분. 사용:

```
throw new SentenceException(ESentenceErrorType.NOT_FOUND, sentenceId);
```

### 2.7 GlobalExceptionHandler

**위치**: `common/exception/GlobalExceptionHandler.java`

컨트롤러에서 발생한 예외를 가로채 표준 에러 응답으로 변환하는 단일 핸들러.

| 항목 | 내용 |
|---|---|
| 어노테이션 | `@RestControllerAdvice` |
| 처리 대상 | `BaseException`, `MethodArgumentNotValidException`, `Exception` |
| 응답 | `ErrorResponse` + `errorType.getStatus()` |

`@RestControllerAdvice` 는 본 클래스 하나만.

### 2.8 ErrorResponse

**위치**: `common/exception/ErrorResponse.java`

클라이언트로 나가는 에러 JSON 구조의 DTO.

```json
{ "error": { "code": "TMI.SENTENCE.NOT_FOUND", "message": "Sentence abc-123 was not found." } }
```

| 항목 | 내용 |
|---|---|
| 종류 | `record` 또는 일반 클래스 (Lombok `@Getter @Builder` 가능) |
| 필드 | `Error error` → `code`, `message` |

## 3. 메시지

**위치**: `src/main/resources/messages/messages.properties`, `messages_ko.properties`

| 파일 | 언어 |
|---|---|
| `messages.properties` | 영문 (기본) |
| `messages_ko.properties` | 한글 |

`{0}`, `{1}` 은 `BaseException` 생성자의 `args` 순서대로 치환.

```properties
TMI.COMMON.INVALID_REQUEST = Invalid request.
TMI.COMMON.INTERNAL_ERROR = Internal server error.
TMI.SENTENCE.NOT_FOUND = Sentence {0} was not found.
TMI.SIMILARITY.REQUEST_INVALID = Invalid request to similarity service.
TMI.SIMILARITY.UNAVAILABLE = Similarity scoring service is unavailable.
```

키 = ErrorType enum 의 `errorCode` 값.
