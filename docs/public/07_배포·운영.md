## 3. 환경 변수

설정값은 코드 외부에서 주입, 환경별 분리.

| 변수 | 용도 | 예시 | 필수 |
|---|---|---|---|
| `SPRING_PROFILES_ACTIVE` | 활성 프로필 (`local` / `prod`) | `local` | Y |
| `DB_URL` | PostgreSQL 접속 URL | `jdbc:postgresql://localhost:15432/tmi` | Y |
| `DB_USERNAME` | DB 사용자 | `tmi_user` | Y |
| `DB_PASSWORD` | DB 비밀번호 | (시크릿) | Y |
| `SIMILARITY_BASE_URL` | Similarity 서버 베이스 URL | `http://localhost:8000` | Y |
| `SIMILARITY_TIMEOUT_MS` | 외부 호출 read 타임아웃 | `2000` | N (기본 2000) |

<!-- 시크릿 관리:
     - 로컬: .env 파일 (gitignore)
     - 배포: 호스팅 환경 시크릿 매니저 -->

## 4. 헬스체크

```
GET /api/v1/health
```

```json
{ "status": "UP", "db": "UP", "similarity": "UP" }
```

| 필드 | 검사 내용 |
|---|---|
| `status` | 전체 상태 (DB·similarity 모두 UP일 때만 UP) |
| `db` | `SELECT 1` 성공 여부 |
| `similarity` | Similarity 서버 `/health` 호출 성공 여부, 200ms 캐시 |
