TMI 학습 사이클

사용자 한 번의 학습 루프 · 4단계 순환

1
출제
선택한 난이도·카테고리에서 영어 문장 1건 랜덤 추출. 모범 답안은 숨김.
System → User
2
번역 입력
사용자가 영어 문장을 보고 한국어로 번역해 제출.
User → System
3
채점
유사도 서버가 사용자 답안과 모범 답안의 의미적 유사도를 점수로 계산.
System (AI)
4
피드백
점수 + 모범 답안 + 사용자 답안 나란히 표시. 다음 문제로 이어짐.
System → User
↻ 4 → 1로 회귀. 사용자가 멈출 때까지 루프 반복.
